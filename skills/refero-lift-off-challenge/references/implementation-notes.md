# Implementation Notes

Source URL: https://styles.refero.design/style/cf1f4666-bb5b-4fc4-a3e6-660218996cbb

## Extracted Identity

- Title: `Lift-off challenge — Style Reference`
- Theme: `light`
- Visual metaphor: aircraft control panel / mission-control console
- Tone: industrial, mechanical, high-stakes, urgent

## Practical Design Translation

- Canvas: `#e5e7eb`
- Display surfaces: `#11161c`
- Main text on light: `#000000`
- Main text on dark: `#ffffff`
- Accent / warning CTA: `#f43325`
- Secondary detail text: `#575c75`
- Muted / inactive states: `#a3a3a3`, `#bbbbbb`

## Typography Direction

- Main UI text: `proxima-nova` or closest strong sans fallback
- Technical labels: `SF Mono` or `monospace`
- Display numerics: `Doto` or a visibly digital substitute

## Shape System

- Dense compact spacing
- 4px base unit
- 8px common card padding
- Extremely rounded pills/buttons where used for CTA
- Otherwise keep small default radii for utilitarian controls

## Component Guidance

- Primary CTA: red pill, white text
- Data cards: black module, white headline, slate secondary text
- Inputs: light control-panel surfaces with strong borders
- Labels: mono, small, technical, slightly spaced
- Status blocks: compact, aligned, high signal

## Anti-Patterns

- No generic purple SaaS palettes
- No airy luxury dashboard styling
- No heavy blur/glass surfaces
- No decorative noise that weakens the engineered feel
